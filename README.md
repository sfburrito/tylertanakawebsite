# Tyler Tanaka Personal Research Site

Simple one-page research website built with Next.js and Tailwind for `tylertanaka.com`.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Publish

The easiest path is:

1. Push this folder to a GitHub repository.
2. Import that GitHub repository into Vercel.
3. Add the `tylertanaka.com` domain in Vercel when you're ready.

## Edit content

Most site content lives in `app/page.js`.
